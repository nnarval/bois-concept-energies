# Lasian — Poêles à granulés
**Marque :** Lasian | **Catégorie :** poeles-a-granules | **Total :** 5 modèles  
**Fabricant :** Lasian Tecnología del Calor S.L. — Muel, Zaragoza (Espagne)  
**Disponibilité BCE :** Tous proposés (`availabilityForBce: "propose"`)

> 🎨 **Sélecteur couleur côté client :** Chaque modèle propose plusieurs finitions. La structure JSON `couleurs[]` contient pour chaque coloris son `slug`, `label`, `hex`, `imageUrl` et `imageUrlBg` — prêts à alimenter un sélecteur interactif qui met à jour l'image produit en temps réel quand le client change de couleur.

## Structure JSON sélecteur couleur (guide développeur)

Chaque produit expose un tableau `couleurs[]` structuré pour le sélecteur interactif :

```json
"couleurs": [
  {
    "slug": "effet-rouille",        // identifiant URL-safe
    "label": "Effet rouille",       // label affiché
    "labelEs": "Efecto óxido",      // label espagnol (fabricant)
    "hex": "#8B4513",               // couleur CSS pour pastille
    "imageUrl": "https://...",      // image principale du poêle dans ce coloris
    "imageUrlBg": "https://...",    // image fond transparent / variante
    "disponible": true              // contrôle l'état du bouton (grisé si false)
  }
]
```

**Usage recommandé :** Au clic sur une pastille couleur, remplacer `imageUrl` principale par `couleurs[selected].imageUrl`. La pastille affiche `hex` comme fond. Le label s'affiche en dessous. Si `disponible: false`, la pastille est grisée et non cliquable.
